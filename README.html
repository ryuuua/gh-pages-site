Plots Gallery (Lightweight)

How this build is generated

  1. Run node scripts/create-public-gallery.js in the main project to generate public_gallery/ and assets/data/gallery-data.public.json.
  2. Copy the resulting public_gallery/ and manifest into this directory (see README.md).
  3. Commit this directory to the repository that backs GitHub Pages.

Only a handful of assets are included so that GitHub Pages stays within quota limits. To expand the gallery, regenerate the subset with different filters before copying.
